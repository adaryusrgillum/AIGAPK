# Abel Insurance Group - Mobile App

A React Native (Expo) mobile application for **Abel Insurance Group**, a family-owned independent insurance agency based in Buckhannon, WV.

## Features

- **Home** – Hero section, insurance solution overview, client testimonials, carrier partners
- **Personal Insurance** – Browse 16+ personal coverage types (Home, Auto, Life, Umbrella, etc.)
- **Business Insurance** – 14+ business coverages with industry-specific tab
- **About** – Company mission, values, licensed states, insurance partners
- **Reviews** – Client testimonials with star ratings
- **Contact** – Phone, email, address, office hours, social links, client portal, and contact form

## Install the APK

Go to [Releases](../../releases) and download `AbelInsuranceGroup.apk`. Enable "Install from unknown sources" on your Android device and open the APK.

## Development

```bash
npm install
npx expo start
```

## Build APK

The APK is built automatically via GitHub Actions on every push to `main`. You can also trigger it manually from the Actions tab.
